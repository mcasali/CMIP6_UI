Choosing File Type

Statistical Downscaling

Shapefile: Shapefiles are very common geospatial data format. It was developed for Esri GIS however its published specifications ensure interoperability with many commercial and open source GIS software products. Due to irregular nature of the gridded CCSM model output this portal distributes the datasets in a point shapefile format, where each point represents a centroid of a corresponding CCSM grid cell.

Text File: Climate projections can be downloaded in a comma-delimited textfile format. This popular and simple data format can be brought into various data analysis packages and spreadsheets.

Geojson: GeoJSON is an open standard geospatial data format that is based on JavaScript Object Notation (JSON).
